## Getting started

### Prerequisites

- A working version of Python>=3.8

### Install

```sh
# ---------- Linux / MacOS ----------
python3 -m pip install depthai-viewer
# ------------- Windows -------------
python -m pip install depthai-viewer
```
### Run
```sh
python3 -m depthai_viewer
# --------  OR  ---------
depthai-viewer
```
