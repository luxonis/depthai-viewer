## Install

```sh
python3 -m pip install depthai-viewer
```

## Run

```sh
depthai-viewer
# OR
python3 -m depthai_viewer
```

## Troubleshooting

You can run with `RUST_LOG=debug` to get more output out of the depthai-viewer.
